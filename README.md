# Contact Book UI (React)

This is a React based UI to work with Contact Book Orchestration.

## Installation

Clone this repo and run:

```sh
npm install
```

## Try it out

```sh
npm run dev:stubs
```

## Production build

The following command will create a full build into the `dist` folder.

```sh
npm run build
```
